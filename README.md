# OpsPulse

**OpsPulse** is a real-time logistics and delivery operations dashboard designed for Ethiopian and global businesses. It provides instant, actionable insights for operations managers, agents, and business owners.

## Project Structure

- `backend/` — FastAPI backend (Python), database, real-time APIs
- `frontend/` — React/Next.js user interfaces for dashboard and agent app
- `docs/` — All system design documents and notes

## Getting Started

Each part of this project is well-documented and built step-by-step, focusing on learning, quality, and best engineering practices.
